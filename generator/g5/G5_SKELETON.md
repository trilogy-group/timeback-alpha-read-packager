## 1. Units → Lessons map (mirrors the G3/G4 band architecture)

Three bands = three units. Within each band, the G3 "5-stage fade" folds in as within-band
checkpoints (Band A = stages 1–2, Band B = stages 3–4, Band C = stage 5) — per G3 Fix B1.
**~60 doses total**, 25 min each. Dose counts mirror G3's `~3 / 5 / 40–48 / 6` split.

| Unit | Doses | Role | Topics (count) | Lexile sub-band | Gates |
|---|---|---|---|---|---|
| **Part 1 — Orientation / on-ramp** | ~3 | how the course works; fixed-opener topic; schema Toolkit soft intro | 1 fixed-opener (+ Toolkit) | ~830–860L | none (advisory) |
| **Unit A — On-ramp expeditions** | ~5 | topic-coherent, support-heavy; schemas exercised, NOT walled | 3 topics | ~830–880L | advisory only |
| **Unit B — Core expeditions** | ~40–48 | the curriculum lever (G5's primary lever); schemas reach enforced mastery inside expeditions; topic-coherence begins to fade mid-band | 4 topics | ~880–970L | **blocking ≥90% gates begin** (per-schema, per-KC) |
| **Unit C — Cold mixed gauntlet** | ~6 | full mixed-topic cold transfer; longest passages (G5 stamina lift) | 3 topics, interleaved | ~970–1010L | final 90% mastery gate |
| **Foundations strand** *(routed)* | as routed | word-attack/vocab on-ramp for below-floor readers | n/a | below band | non-blocking | 

> Lesson granularity inside a unit (mirror G3): each topic = an **expedition** of ~4–6 daily
> doses; each dose = a 6-step micro-cycle **hook → teach → guided → read → check → CFU**
> (Word-Assist on steps 1–4, OFF on 5–6). Each KC or schema is one atom inside a dose. A dose
> "counts done" only when the check is *attempted* (honest completion).
>
> **G5 note vs G3:** G5 is fully "reading to learn" — decoding assumed. The Foundations
> on-ramp is likely **smaller or absent** at G5; size it from the **[TO-SOURCE]** G5 cohort
> data, do not carry G3's Foundations weight over.

## 3. Where each component slots (the assembly map)

| Component | Items | Slots into | Gating | Source bank |
|---|---|---|---|---|
| **Content lessons** | units × lessons × ~15 items (G3/G4 structure) | inside each expedition dose (hook→…→CFU); KCs + schema atoms | **non-gating** (progress never blocked) | `g5_demo_bank.json` `component:"content"` |
| **Mastery gate** | **24-item** form × **3–4 parallel forms**; staged 90% | end of Band B (per-schema, per-KC) + final at course end; powerpath-100 parent/child ALIs | **ENFORCED** (≥90% on unseen; near-neighbour) — **cold-transfer items folded in** (Unit C) | `component:"mastery"` |
| **MAP-equiv test** | **40 items × 2 forms**; each item its OWN cold passage | held-out, taken outside expeditions (pre/post); never taught-to, never gated-on | **non-gating**; raw/40 + band; **NO RIT** | `component:"map_equiv"` |
| **Spaced review** | 3 bands × ~15; ≥50% weak-skill, interleaved (FSRS) | between expeditions; re-tests Developing KCs on fresh passages | non-gating | `component:"spaced"` |
| **Transfer Check** | cold-passage set, all legal formats | Unit C (mixed cold gauntlet) + readiness checks | non-gating | `component:"transfer"` |

**Retry routing (every gate):** 3 states — **✓ Mastered** (≥90% on unseen + holds on later
spaced re-check → advance/certify) · **◐ Developing** (cleared a lower threshold → keep moving,
KC stays on FSRS queue, re-tested on fresh passages) · **✗ Not-yet** (diagnose → re-teach in a
different modality → retry on a *parallel same-difficulty* passage → after 2 cycles escalate to
step-down/Foundations/human). **Never re-serve the identical item; never hand a harder item
after failure.** (G3 Insight 4.)

### 3a. MAP-equiv 40-item blueprint (ported to G5 — canonical mix)

Content: **Literary 35% (14) / Info 32% (13) / Vocab 33% (13)**.
Difficulty: **easy 6 / medium 14 / hard 20** (centered above grade).
Format: **MCQ 24 / EBSR 3 / MSQ 4 / hot-text 5 / match 4** (16 TEI = 40%).

| | easy | medium | hard | total |
|---|---|---|---|---|
| MCQ | 3 | 9 | 12 | **24** |
| EBSR | 3 | 0 | 0 | **3** |
| MSQ | 0 | 2 | 2 | **4** |
| hot-text | 0 | 2 | 3 | **5** |
| match | 0 | 1 | 3 | **4** |
| **total** | **6** | **14** | **20** | **40** |

Per-domain (ported from G3 `DESIGN.md` §2.4): Literary = 5 MCQ + 2 EBSR + 2 MSQ + 3 hot-text +
2 match; Informational = 6 MCQ + 1 EBSR + 2 MSQ + 2 hot-text + 2 match; Vocabulary = 13 MCQ.
The 6 easy items = **3 EBSR + 3 MCQ** (EBSR easy-band-only). Each item = its own cold passage.
**Band cut scores + their RIT meaning are [TO-SOURCE] / [PENDING PILOT]** — report raw/40 + band.

### 3b. Mastery-gate 24-item blueprint (ported to G5)

24-item form, **3–4 parallel forms**, pass ≥90% on unseen passages, near-neighbour distractors
required, no FRQ (MAP is 100% auto-scored). Domain inclusion guaranteed; format mix follows the
legal-G5 set (mirror G3 `mastery_test_blueprint` `_format_split`, adjusted to drop `order`).
`per_item_p_m_min` is a **TARGET to measure post-pilot, NOT a property we ship** (G3 Insight 6).

## 4. Lexile progression: 830L → 1010L across the course

G5 CCSS stretch band ≈ **830–1010L** (spec §A.4 / brainlift — **TO-CONFIRM against Alpha's
internal target**). Passages run **~120–160 words** (median near 140; reserve 150–160 for the
hard band only — `edullm-ela` ~140-word soft cap). The staircase is a *personal* one (G3
Engine 4), but the course-level reference progression is:

| Stage | Unit | Lexile sub-band | Passage length | Notes |
|---|---|---|---|---|
| Open | Part 1 + Band A early | **~830–860L** | ~120–135 w | on-ramp floor; topic-coherent; support-heavy |
| Band A late | Unit A | ~860–880L | ~130–140 w | schemas exercised, advisory checks |
| Band B early | Unit B | ~880–930L | ~140 w | enforced gates begin; denser info text |
| Band B late | Unit B | ~930–970L | ~140–150 w | topic-mixing phases in; harder inference |
| Band C | Unit C | **~970–1010L** | ~150–160 w | cold mixed gauntlet; longest literary blocks (stamina lift) |

- **Difficulty ≠ Lexile alone.** Lexile is the readability proxy; the MAP-equiv `hard` band
  (700–820L at G3) maps up to the **upper G5 band ~950–1010L** but item difficulty is also set
  by relational keys + near-neighbour distractors (G5 generator rule 1: prefer relational keys).
- **RIT↔Lexile is [SECONDARY] / [PENDING PILOT]** — do NOT print a RIT staircase. The
  ~20 L/RIT planning estimate that G3 used is G3-specific; G5 RIT meaning waits on the pilot.
- Each MAP-equiv item carries its own cold passage at its tagged sub-band; lesson/mastery items
  may share a passage *within a lesson* but that passage appears **nowhere else** in the course.
